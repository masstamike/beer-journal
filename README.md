# beer-journal
Review the flavors that make your favorite beer great in a personal beer journal.

A simple journal to keep information such as:
  * Brewery
  * Name of Beer
  * Price Point
  * Date Sampled
  * Rating
  * Serving Type (draft, bottled, etc.)
  * Stats (ABV, IBU, etc.)
  * Flavor aspects (spicy, herbal, malty, etc.)
